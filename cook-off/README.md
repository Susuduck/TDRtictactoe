# Cook-Off: Kitchen Chaos

A fast-paced order matching game where players must tap ingredients in the correct order to complete recipes before time runs out.

## How to Play

1. **Match Orders**: Orders appear showing required ingredients in a specific sequence
2. **Tap Ingredients**: Select ingredients from the available pool in the exact order shown
3. **Beat the Timer**: Each order has a countdown timer that decreases as the game progresses
4. **Avoid Mistakes**: 3 wrong orders result in game over
5. **Build Combos**: Complete consecutive orders without mistakes for bonus points

## Game Features

- **10 Unique Opponents**: From Funky Frog to Grand Master Grizzly, each with their own cooking challenge
- **10 Levels Per Opponent**: Increasing difficulty with faster timers and more complex recipes
- **Star Progression System**: Earn points to unlock stars (4 points = 1 star, 40 points to master)
- **Sequential Unlocking**: Master previous opponents to unlock new challenges

| Disco Dinosaur | Tap on beat for score multiplier |
| Radical Raccoon | Wild card ingredient appears |
| Electric Eel | Occasional speed rounds |
| Mysterious Moth | One ingredient is hidden initially |

| Sly Snake | Watch out for decoy ingredients |
| Wolf Warrior | Multiple orders at once |
| Grand Master Grizzly | All challenges combined! |

## Controls

- **Mouse/Touch**: Tap ingredients to add them to the order
- **ESC**: Return to previous menu

## Scoring

- Base points for completing orders
- Time bonus for quick completion
- Combo multiplier for consecutive orders
- Special bonuses for rhythm timing and speed rounds

## Progress

Progress is automatically saved to localStorage (`cookoff_progression_v1`).
