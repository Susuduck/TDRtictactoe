# Shooting Gallery - Pest Blaster

## Overview
A fast-paced target shooting game where bugs pop up and you must blast them before they disappear. Each opponent introduces NEW mechanics, not just faster spawns.

## Design Philosophy
- **Pattern Learning**: Targets spawn in learnable patterns (formations with Professor Penguin)
- **Combo System**: Builds upon skill - chain hits for bonus points
- **Clear Feedback**: Immediate visual feedback for hits, misses, and combos
- **Agency**: Strategic target selection (high value vs safe)
- **New Mechanics Per Opponent**: Each arena teaches something new

## How to Play
1. Click or tap on bugs to shoot them
2. Each bug type has different point values
3. Chain hits for combo multipliers
4. DON'T shoot the friendly Worm Gang members!
5. 60 seconds per round

## Controls
- **Click/Tap**: Shoot at that position
- **ESC**: Return to previous screen

## Scoring
- Base points from bugs
- **Combo Bonus**: 5+ chain = +10% per combo
- **Rhythm Bonus**: 2x on the beat
- **Chain Lightning**: +20 for chained bugs

## Win Conditions
- Target score scales with level and opponent
- 50%+ of target = Pass (+1 point)
- 80%+ of target = Excellent (+2 points)

## Tips
1. Build combos early - they snowball
2. Learn spawn patterns for Professor Penguin
3. Rhythm bugs glow on the beat - time your shots!
4. In spotlight mode, let bugs come to the light
5. Boss bugs are worth waiting for - 200 points!
6. Chain lightning bugs can clear clusters

## Why This Game is Fun
- **Mastery Curve**: Each opponent teaches a new skill
- **Clear Feedback**: You always know why you hit or missed
- **Strategic Depth**: Target prioritization matters
- **Flow State**: Adaptive spawn rate keeps you engaged
- **Variety**: No two arenas play the same way
